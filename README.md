# 🚨 AI Incident Investigator

An AI-powered software log analysis application built with **Python**, **Streamlit**, and **Google Gemini AI**.

The application analyzes uploaded software log files, detects incidents, identifies possible root causes, classifies severity, and generates AI-assisted investigation reports.

---

## 📌 Features

- 📂 Upload `.log` and `.txt` files
- 📊 Log statistics dashboard
- 🚨 Incident severity detection
- 📈 Incident distribution chart
- 🕒 Incident timeline
- 🔍 Possible root cause identification
- 🤖 AI-powered incident analysis using Google Gemini
- 📄 Download incident report

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- Pandas
- Matplotlib

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/laxmiprasannapolagowni/AI-Incident-Investigator.git
```

Go into the project folder:

```bash
cd AI-Incident-Investigator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
