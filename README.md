# 🚨 AI Incident Investigator

An AI-powered log analysis application built with **Python**, **Streamlit**, and **Google Gemini AI**.

The application analyzes uploaded software log files, detects incidents, identifies possible root causes, classifies severity levels, and generates AI-assisted investigation reports.

---

## ✨ Features

- 📂 Upload `.log` and `.txt` files
- 📊 Log statistics dashboard
- 🚨 Automatic incident severity detection
- 📈 Incident distribution visualization
- 🕒 Incident timeline
- 🔍 Possible root cause identification
- 🤖 AI-powered analysis using Google Gemini
- 📄 Download investigation report

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- Pandas
- Plotly
